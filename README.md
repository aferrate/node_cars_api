# Sample project API with Nodejs and Docker
### Install

run docker:
```
 docker-compose up
```


### Start application

access phpmyadmin:
- [http://localhost:8081](http://localhost:8081)

credentials:
```
 user root
 password root
```

call localhost in your browser to test if works:
- [http://localhost:3000/api/v1/](http://localhost:3000/api/v1/)
